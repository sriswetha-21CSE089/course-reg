About this course

You will find learning CSS essential in styling websites. Web designers use it to build on basic HTML and add personality to plain text pages. This course helps you expand your coding foundation and gives you CSS interactive practice to start adding colors and background images or editing layout so you can create your very own ,unique stylized web pages.

Skills you'll gain

  • Add styling to HTML elements
  • Connect HTML and CSS
  • Create unique web page layouts
