About this course

Thanks to a growing number of software programs, it seems as if anyone can make a webpage.But what if you actually want to understand how the page was created? There are great textbooks and online resource for learning web design.

Description

HTML is everywhere and more and its more important than ever.Its not only for web developers, having knowledge of HTML and how it works is beneficial in most careers.

HyperText Markup Language,commonly abbreviated as HTML, is the standard markup language used to create web pages as well as to create user interfaces for mobile and web applications.

By the end of course you will have the skills and know how to apply HTML to make a real website.
